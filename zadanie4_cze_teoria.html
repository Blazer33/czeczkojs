Pytania teoretyczne i odpowiedzi

1) Kiedy powstał JavaScript (podaj datę).
JavaScript został stworzony w 1995 roku.

2) Dlaczego jest to język skryptowy?
JavaScript jest językiem skryptowym, ponieważ jest interpretowany w czasie rzeczywistym przez przeglądarkę, co pozwala na dynamiczną interakcję z użytkownikiem.

3) Gdzie jest wykonywany JS (klient lub serwer)?
JavaScript jest wykonywany głównie po stronie klienta.

4) Dlaczego JS to bezpieczne narzędzie?
JavaScript ma ograniczenia dostępu do systemu użytkownika, co czyni go bezpiecznym do wykonywania na stronach internetowych.

5) Czy jest to język obiektowy? Jeśli tak, to dlaczego.
Tak, JavaScript jest językiem obiektowym, ponieważ pozwala na tworzenie i manipulowanie obiektami.

6) Wymień zalety JS:
Zaleta 1: Szybki czas wykonania.
Zaleta 2: Możliwość tworzenia interaktywnych stron.
Zaleta 3: Obsługiwany przez wszystkie przeglądarki.

7) Framework to:
Framework to zbiór gotowych bibliotek i narzędzi ułatwiających tworzenie aplikacji.

8) Do aplikacji mobilnych można użyć Framework
Tak, istnieją frameworki do tworzenia aplikacji mobilnych, np. React Native.

9) Aplikacje responsywne to:
Aplikacje, które dostosowują swój wygląd do rozmiaru ekranu urządzenia.

10) Przepisz linię kodu poniżej, a następnie wytłumacz, jak działa ten fragment kodu:
<input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')">
Ten kod tworzy przycisk, który po kliknięciu wywołuje funkcję `WinOpen`, otwierając nowe okno.

11) Przepisz linię kodu poniżej, a następnie wytłumacz, jak działa ten fragment kodu:
window.open("obraz.html", "okienko", "toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");
Otwiera nowe okno o nazwie "okienko" z plikiem `obraz.html` w ustalonym rozmiarze i pozycji na ekranie.

12) Instrukcja window.close()
Zamyka aktualne okno przeglądarki, w którym jest wywoływana.
